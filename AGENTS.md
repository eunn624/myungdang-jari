# 명당자리 — Codex 프로젝트 가이드

## 📋 프로젝트 개요

**프로덕트:** 사주/풍수 기반 초개인화 명당 가이드 & 개운 앱  
**한 줄 정의:** 사주 오행을 기준으로 *어디서 살고, 어떻게 배치하고, 무엇을 둘지*를 큐레이션하는 라이프스타일 앱  
**포지셔닝:** 라이프스타일·엔터테인먼트 (무속색 배제, 효능 보장 표현 금지)  
**현재 페이즈:** Phase 1 MVP (입력폼 · 만세력 · 행정동 매칭 · 지형 가이드 · 침대 방향 · 아웃링크)

---

## 🎯 핵심 기능 (Phase 1)

### 1. 명당 분석 엔진
- **입력:** 생년월일시 (+ "시 모름" 옵션)
- **처리:** 만세력 라이브러리로 오행(木·火·土·金·水) 계산 → **클라이언트 단에서만 수행**
- **출력:** 부족 오행을 보완하는 행정동 추천 (지명·지형 기반)
  - 예) 水 필요 → '水·川·江·海' 포함 행정동 + 수변 인접 행정동

### 2. 침대 머리 방향 가이드
- 사주 기준 길한 방향(예: 동쪽) 텍스트 안내
- **나침반/센서 기능 없음** (단순 방향 안내만)
- 구조상 어려울 때의 대체 인테리어 팁 제공

### 3. 개운 커머스 큐레이션
- 오행별 맞춤 상품 피드 (쿠팡 파트너스·오늘의집 등)
- **제휴 관계·대가성 표기 필수** (법규 준수)

### 4. 데일리 개운 루틴 (리텐션 엔진)
- 일진·컬러 등 데일리 카드
- 개운 미션 (예: "오늘은 동쪽에 식물 두기")
- 푸시 알림 (옵트인)

## 📊 정보 구조 & 플로우

### 온보딩 (첫 진입)
```
스플래시 → 가치 소개 → 개인정보 동의 
→ 생년월일시 입력(+'시 모름 옵션') 
→ 오행 분석 로딩 → 결과 홈
```

### 코어 루프 (재방문)
```
데일리 카드 확인 
→ 개운 미션 
→ 침대 가이드/지명 재확인 
→ 맞춤 상품 클릭 
→ 결과 공유
```

### 글로벌 네비게이션 (탭)
- **홈** (데일리)
- **명당** (지명·지형·침대)
- **스토어** (큐레이션)
- **마이** (프로필·궁합)

## 🔧 기술 스택 & 아키텍처

### 플랫폼
- iOS / Android (Expo + React Native)

### 핵심 라이브러리
- **만세력 계산:** `saju-calculator` 또는 동등 만세력 라이브러리
  - ⚠️ **정확도 주의:** 진태양시 보정·자시 경계·윤달 처리를 **하나의 기준으로 고정**하고 전문가 검증 필수
- **음력/양력 변환:** 내장 또는 `lunisolar` 등
- **상태 관리:** Redux 또는 Zustand
- **API:** Express.js (Node) 또는 Next.js

### 아키텍처 원칙
1. **개인정보 보호:** 생년월일(시) 계산을 **클라이언트에서만 수행** → 서버는 매핑 테이블·상품·콘텐츠만 제공
2. **성능:** 결과 산출은 수 초 내, 캐싱 활용
3. **행정동 기준:** GIS 정밀도 제거 → 행정동 단위로 통일
4. **런타임 최적화:** 지형 속성은 **사전 계산 후 조회** (오프라인 테이블화)

## ⚠️ 중요 제약 & 포지셔닝

### 포지셔닝 준수 사항
- ✅ 라이프스타일·엔터테인먼트로만 표현
- ❌ "운이 트인다", "병이 낫는다" 등 효능/결과 보장 표현 금지
- ✅ 모든 결과에 "오락·참고 목적" 면책 고지 필수

### 법규 준수 (Phase 1 Exit 전 검토)
- 개인정보 수집·이용 목적·보관기간 명시 동의
- 제휴 커머스 아웃링크에 대가성·제휴 관계 표기
- 연령 제한 (만 14세 미만 법정대리인 동의)

## 🎨 디자인 원칙

- **무드:** 무속색 배제, 미니멀·세련된 매거진 스타일
- **색상:** 톤다운 뉴트럴 + 자연물(물·돌·나무) 그래픽
- **폰트 안정성:** 영문/숫자 혼용 시 이탤릭 서브패밀리 보유 필수 (가짜 기울임 방지)
- **접근성:** WCAG AA 명암비, 동적 글자 크기, 스크린리더 레이블

## 📝 Personas

### Persona A: 자취생 (1~2년 주기 이사)
- Needs: 침대 방향 안내, 저가 비보 소품, 대안 솔루션

### Persona B: 예비 매수자 (신혼/생애 첫 내 집)
- Needs: 부부 궁합 기반 동네 안내, 거시 지형 가이드

### Persona C: 개운 루틴형 3040 직장인 *(리텐션 핵심)*
- Needs: 데일리 일진·컬러, 시즌별 소품, 반복 재방문
