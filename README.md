# 🔮 명당자리 — 사주/풍수 기반 초개인화 명당 가이드 & 개운 앱

> 나의 운을 트이게 하는 공간 가이드

**프로젝트 상태:** MVP 개발 중 (Phase 1)  
**문서:** [PRD v3.2](./명당자리_PRD_v3.2.md)

## 🎯 핵심 기능

- 🎂 **생년월일시 입력** → 사주 오행 분석
- 🏘️ **행정동 명당 추천** → 지명·지형 기반 매칭
- 🛏️ **침대 머리 방향 가이드** → 텍스트 안내
- 🛍️ **개운 커머스 큐레이션** → 아웃링크 제휴
- 📅 **데일리 개운 카드** → 일진·미션·푸시 알림

## 🚀 빠른 시작

### 설치
```bash
npm install
```

### 개발 서버 시작
```bash
npm run dev
```

### 빌드
```bash
npm run build
```
